## Description
This is a pre-trip packing checklist web application that helps you stay organized before every journey.You can customize lists based on your destination or trip type and check off items as you pack, ensuring you never forget essentials and making travel preparation simple and stress-free.  
![Demo](https://travel-list-lime-nine.vercel.app/) 

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
